If every object properly contained in x is properly contained in y, x is contained in y

\begin{align*}
((\exists z)\\
& properContainedIn(z,x) \wedge (\forall z) (properContainedIn(z,x)\\
&\supset properContainedIn(z,y))\\
&\supset containedIn(x,y)
\end{align*}
